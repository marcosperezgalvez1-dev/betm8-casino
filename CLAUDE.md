# Casino Online - Memoria del Proyecto

## Descripcion del Proyecto
Web de casino online con dinero simulado para jugar en salas privadas con amigos.
No se usa dinero real. El objetivo es crear una experiencia de casino elegante y divertida
donde grupos de amigos pueden crear salas privadas y jugar juntos.

## Stack Tecnologico
- **Frontend:** HTML5, CSS3, JavaScript (vanilla, sin frameworks)
- **Backend:** Python con Flask (pendiente)
- **Base de datos:** MySQL (pendiente)
- **Control de versiones:** Git + GitHub
